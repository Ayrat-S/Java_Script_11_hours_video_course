*** 07:38 Как выполнять код

Где же выполнять джс код во время этого курса?
Все будет выполняться на одном вашем компьютере и нескольких программ, которым вы уже пользуетесь.
Вы можете выполнять код в веб браузерах - гугл хром самый популярный браузер и я совтую использовать именно его.
Джс встроен в любой веб браузер, поэтому вам ничего дополнительного устанавливать не нужно - просто использовать свой веб браузер.
Также вы можете запускать джс код используя Node.js - это среда выполнения джс кода. Если нод джс не установлен, установите дополнительно.
Кроме того, я вам советую использовать редактор кода - Visual Studio Code, это программа полностью бесплатная и писать код в редакторах кода наного более удобно нежели в обычном текстовом файле или ворде. Потому что редакторы кода специально созданы для того чтобы писать код, и у них есть функции подсветки разных частей кода, автозавершение разных команд, которые вы пишете на том или ином языке программирования, а также, если есть возможность установки расширений, в частности в редакторе Visual Studio Code есть такое расширение как Code Runner - с помошью этого расширения можно запускать различные блоки кода, написанные на разных языках, в том числе на джс.
