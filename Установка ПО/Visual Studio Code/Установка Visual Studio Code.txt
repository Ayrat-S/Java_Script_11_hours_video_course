***УСТАНОВКА Visual Studio Code.
1. Установить редактор кода Visual Studio Code на ПК:
В веб браузере введите
Visual Studio Code
и перейдите по первой ссылке
code.visualstudio.com
затем установите на ваш ПК.

2. Чтобы выполнять, вам нужно установить расштрение Code Runner.
Для этого в Visual Studio Code, в левом вертикальном меню, выбираем иконку
Extensions
и находим в поисковике расширение Code Runner (оранжевый .run / Автор Jun Han).
Выбираем, устанавливаем через Install, после чего в правом верхнем углу появится кнопка запуска кода Run Code в виде треугольника.
Установленные расширения можно найти под строкой поиска - нажать на всплывающий каталог INSTALLED.
