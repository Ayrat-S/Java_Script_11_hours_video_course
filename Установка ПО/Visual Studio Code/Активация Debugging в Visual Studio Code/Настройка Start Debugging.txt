***НАСТРОЙКА Start Debugging (F5).
1. Теперь сделаем следующее, перйдите на первый пункт вертикального меню слева - Explorer, и здесь же создадим новый файл.
Кликните правой кнопкой мыши и нажмите New file.
Здесь вы можете выбрать язык программирования для вашего файла (кликнуть по активному тексту Select a language) - в нашем случае JavaScript.

2. Чтобы до запуска кода, предварительно установить конфигурацию отладчика, нужно через пункт верхнего горизонтального меню:
Run => Add configuration...
Будет предложен выбор отладчика (Select debugger), например:
Node.js и др..

3. Перед тем как запустить код, нужно нажать "Start Debugging (F5)" через пункт верхнего горизонтального меню:
Run => Start Debugging (F5).
Если отладчик не выбран изначально, то автоматически будет предложен выбор отладчика (Select debugger), например:
Node.js и др..
